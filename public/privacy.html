Kebijakan Privasi Tebar.io

Terakhir diperbarui: 12 Juli 2026

Tebar.io ("ekstensi") adalah alat bantu yang berjalan di WhatsApp Web untuk menyusun dan mengirim pesan broadcast ke kontak Anda. Kami memproses data seperlunya agar fitur berfungsi dan menjaga privasi Anda.

1. Data yang diproses di perangkat Anda (lokal)

  • Draf kampanye, template pesan, pengaturan, dan riwayat laporan pengiriman disimpan secara lokal di browser Anda. Data ini tidak dikirim ke server kami.
  • Kontak dan grup WhatsApp Anda dibaca dan diproses langsung di dalam browser untuk menampilkan pilihan penerima.

2. Data yang dikirim ke server kami

  • Untuk memverifikasi lisensi/akses akun, ekstensi menghubungi server kami dan mengirim informasi akun/lisensi seperti pengenal akun (mis. alamat email atau ID akun), pengenal perangkat/instalasi, dan status lisensi.
  • Saat Anda melakukan pengiriman pesan (Broadcast), ekstensi akan mengirimkan nomor WhatsApp pengirim, nomor penerima, dan isi pesan ke server kami. Data ini diperlukan untuk mencatat riwayat pengiriman dan menghitung batas kuota (limit) sesuai lisensi Anda.
  • Data ini dipakai semata untuk mengelola akses/langganan, mencatat riwayat pengiriman Anda, dan mencegah penyalahgunaan.

3. Yang tidak kami lakukan

  • Kami tidak menjual atau mentransfer data Anda ke pihak ketiga.
  • Kami tidak menggunakan data Anda untuk iklan.
  • Kami tidak menggunakan data untuk menentukan kelayakan kredit atau pinjaman.
  • Kami tidak menggunakan data untuk tujuan di luar fungsi utama ekstensi.

4. Penyimpanan pihak ketiga

Kami menggunakan Supabase sebagai penyedia backend untuk data lisensi/akun. Data tunduk pada standar keamanan Supabase.

5. Penyimpanan & retensi

  • Data lokal tersimpan di perangkat Anda hingga Anda menghapus ekstensi atau membersihkannya.
  • Data akun/lisensi disimpan selama akun Anda aktif.

6. Hak Anda

Anda dapat meminta penghapusan data akun Anda dengan menghubungi kami melalui email di bawah.

7. Penggunaan yang bertanggung jawab

Ekstensi ditujukan untuk mengirim pesan hanya kepada penerima yang telah menyetujui (opt-in). Pengguna bertanggung jawab mematuhi Ketentuan Layanan WhatsApp serta hukum anti-spam dan perlindungan data yang berlaku.

8. Afiliasi

Tebar.io adalah produk pihak ketiga dan tidak berafiliasi dengan, tidak didukung oleh, dan bukan bagian dari WhatsApp LLC atau Meta Platforms, Inc.

9. Perubahan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan akan dipublikasikan di halaman ini.
